MultiNFT

A Comprehensive Multimodal Dataset for NFT Price Prediction and Market Analytics

About

MultiNFT is a comprehensive multimodal dataset of non-fungible tokens (NFTs), sourced from OpenSea. To ensure data accuracy and consistency, the dataset is constructed using official APIs and adopts a three-stream data acquisition architecture that integrates sales records, asset metadata, and standardized image processing.


Spanning a four-year period from 1 May 2021 to 30 April 2025, MultiNFT captures 2.38M transactions involving over 500K unique NFT assets. By encompassing multiple market cycles, the dataset offers a rich foundation for research in NFT price prediction, multimodal learning, and digital asset analytics.

Dataset Description

50 PFP Collections

collections.csv contains basic information for 50 carefully selected PFP collections from OpenSea's top 100 trading volume collections, all created between April 2021 and March 2023. NSFW assets are excluded from the dataset to ensure data quality and compliance.

Images & Attributes

The dataset provides a comprehensive collection of NFT visual content and metadata. Each of the 523,020 assets has a corresponding high-quality image and detailed attribute information, enabling both computer vision and structured data analysis. Images are standardized for consistency across all collections, while attributes preserve the unique trait characteristics that define each NFT's rarity and value proposition.

File Organization

Images: Organized by collection in images/{collection}/{identifier}.png format (e.g., 1.png, 2.png, ...). All images are uniformly processed to 224×224 pixel RGB PNG format for computer vision applications.

Attributes: Stored as attributes/{collection}_attributes.json with structure {'identifier': [list of trait_types and values], ...} providing trait type and value pairs for each asset.

sales_cleaned.csv filters transactions to four major payment tokens (ETH, WETH, DAI, USDC), removing 97 transactions with other tokens, and integrates Yahoo Finance daily close prices to calculate USD values and logarithmic prices for enhanced price analysis.
